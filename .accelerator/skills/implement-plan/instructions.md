Never include code comments that describe what code could otherwise express. 
Comments only provide value if they signal something extremely non-obvious to
the reader. We have a *very* low tolerance for comments. Actively remove them
from any code you create or modify. Further, references to ADRs, work items, 
ACs or other meta documents in comments can go stale fast so don't include 
them / actively remove them.
